# Моя Волга 2026 — генератор сценариев

Этот репозиторий содержит OpenCode skill для автоматической генерации сценариев 5-минутных видеорепортажей экспедиции «Моя Волга 2026».

---

## Структура

```
.opencode/
└── skills/
    └── volga-story/
        ├── SKILL.md                        # Инструкции для модели
        └── reference/
            └── scenario-template.md        # Типовой шаблон раскадровки
```

---

## Использование в OpenCode

### 1. Установить модель

Skill работает с любой моделью, можно попробовать бесплатную модель **Big Pickle** через OpenCode Zen.

### 2. Открыть проект в OpenCode

```sh
cd /path/to/volga-story
opencode
```

Skill `volga-story` будет обнаружен автоматически из `.opencode/skills/volga-story/SKILL.md`.

### 3. Запросить сценарий

Напиши в чате:

```
Создай сценарий. Мы в Болгарах, вышли из Тетюш, идём в Казань. День 6. На борту: Александр, Алексей, Евгений, Миронов.
```

Модель загрузит skill и сгенерирует полный сценарий по шаблону — с раскадровкой, текстами, вопросами для интервью и чек-листом.

Если не указать тему — модель предложит 2–3 варианта, подходящих к точке маршрута и составу экипажа.

## Reference

Файл `.opencode/skills/volga-story/reference/scenario-template.md` содержит типовую раскадровку с таблицами таймингов, ролями участников и планом действий на день съёмки. Модель может обращаться к нему при генерации.

## Маршрут

Самара → Тольятти → Ульяновск → Болгары → Казань → Свияжск → Чебоксары → Нижний Новгород → Плёс → Кострома → обратно → Самара.

Сроки: 17 апреля — 17 мая 2026 года (~30 дней, ~2220 км).
